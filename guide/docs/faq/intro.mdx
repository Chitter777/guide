# Часто задаваемые вопросы

Это список часто задаваемых вопросов, касающихся использования `disnake` и его дополнений. Не стесняйтесь предлагать
что-то новое на [сервере поддержки](https://discord.gg/disnake), или отправьте его с помощью запросов на добавление
в [репозитории](https://github.com/DisnakeDev/guide/pulls).

## Легенда

-   `bot` ссылается на экземпляр <DocsLink ext="commands" reference="disnake.ext.commands.Bot">commands.Bot</DocsLink>.
    Например, `bot = commands.Bot(command_prefix="!", ...)`.
-   `ctx` ссылается на экземпляр <DocsLink ext="commands" reference="disnake.ext.commands.Context">commands.Context</DocsLink>,
    в то время как `inter` ссылается на <DocsLink reference="disnake.Interaction">disnake.Interaction</DocsLink>.
    Например, `async def command(ctx: commands.Context, ...):`
